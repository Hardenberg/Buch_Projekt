# Kapitel 7 – Redundanzfehler in perfekter Symmetrie

Jonas ließ beide Storage-Cluster gegeneinander laufen – Block für Block, Hash für Hash. Zwei Systeme, getrennt, redundant, gebaut, um Fehler zu entlarven.

Die Abweichungen tauchten paarweise auf.

Linker Cluster: Offset 0x01A3F.
Rechter Cluster: Offset 0x0E5C0.

Er zog die Werte ab. Gleiche Distanz zum jeweiligen Blockende. Spiegelverkehrt.

Er wiederholte den Vergleich mit weiteren Blöcken. Immer dasselbe: keine Streuung, keine Häufung, sondern eine saubere, wiederkehrende Symmetrie, als würde jemand die Fehler so setzen, dass sie sich gegenseitig bestätigen.

Jonas exportierte die Offsets, sortierte sie, visualisierte sie als einfache Punkte.

Aus Zufall wurde eine Linie.
Aus Linie wurde eine Form.

Und diese Form sah aus wie Absicht.
